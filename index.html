jQuery Date Ribbon

Plugin To Create Date Ribbon

View project onGitHub

Description

This jQuery plugin creates date ribbon for your website with full customization.

Current version: 1.1

Download this plugin and use it to display date in a new style on your website.

Features

  • Fully customized
  • Tested in: jQuery 1.4.3+ in Firefox, Safari, Chrome, Internet Explorer 9,10 & 11 Opera
  • Lightweight: 7KB of JavaScript (less than 3KBs gzipped).
  • Load before and load after callback

Instructions & Help

First read usage of plugin (see below). If you found any issue or want any help submit it at GitHub or email me yoku2010@gmail.com

Usage

Date Ribbon accepts setting in Json format, it can be assigned to any HTML element.

// Format:
$(selector).dateRibbon({key:value, key:value, key:value});

// Examples:
$("#ribbon1").dateRibbon({
			title: "GitHub",
			className: "ribbon",
			href:"https://github.com/yoku2010/jQueryDateRibbon",
			target:"_blank",
			date:new Date()
		});
$("#ribbon2").dateRibbon({
			titleFormat:"days"
			className: "ribbon",
			href:"https://github.com/yoku2010/jQueryDateRibbon",
			target:"_blank",
			date:new Date(2012,8,12)
		});

Settings

Property Default Description
title null Define the title of the date ribbon.
titleFormat days The title Format . Can be set to "days", "weeks", "months", or "years".
className ribbon Class name of date ribbon
href # Date ribbon href location or JavaScript function name
target null href location target can be set to "_blank", "_self","_parent","_top" or "frame-name"
date new Date() Date to show in date ribbon
functionBefore function () { return true; } To execute a function just before date ribbon loaded. If function returns true, ribbon will loaded otherwise it will not.
functionAfter function () { } To execute a function just after date ribbon loaded.
